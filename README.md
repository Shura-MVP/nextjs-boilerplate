# شــورى — SHURA MAG

> **المحرّك المعرفي السيادي لمركز دعم اتخاذ القرار**

منظومة استشارية متعددة الوكلاء، تُجري مداولةً منهجيةً مدعومةً بالأدلة، لرفد صنّاع القرار في مركز دعم اتخاذ القرار باستراتيجياتٍ قطاعيةٍ موثّقة.

---

## الفلسفة

شــورى ليست أداة "إجابة"، بل منظومة **مداولة منهجية**. لكل سؤال استراتيجي، تستدعي المنظومة مجلساً من الوكلاء المتخصصين، يحلّل كلٌّ منهم من زاوية تخصصه، ثم تخضع المداولة لتحدٍّ عدائي ومراجعة معرفية، قبل أن يُؤلَّف الموجز التنفيذي النهائي.

**المبادئ الجوهرية:**

| المبدأ | التطبيق |
|--------|---------|
| **الاستشهاد قبل الادعاء** | كل ادعاء جوهري يحمل وسم `[cite:]` أو `[reasoning:]` أو `[gap:]` |
| **تعدد الزوايا** | 14 وكيلاً موضوعياً + 2 عدائي + ناقد معرفي + محرّر موازن |
| **الموازنة قبل الترجيح** | الآراء المعارضة تُوزن، لا تُلغى |
| **الإفصاح عن الفجوات** | ما لا تعرفه المنظومة يُذكر صراحةً |
| **التصنيف السيادي** | كل مداولة تحمل مستوى تصنيف من خمسة |
